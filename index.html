faq

What is PeakAI?

A cross-platform AI agent with a Rust-powered backend. Works with any LLM—run locally with Ollama or connect to OpenAI, Anthropic, Google, and more.

How does it work?

Install the app, configure your preferred LLM (local or cloud), and interact via a beautiful desktop UI or Telegram.

Which models can I use?

All major providers supported: OpenAI, Anthropic, Google, local Ollama, and any OpenAI-compatible API. Works with the latest models.

Is my data safe?

Your data stays on your machine. When using cloud models, only the prompt is sent to the provider—no code or files leave your device.
