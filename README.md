# PeakAI

Open source AI agent for developers.

your code. your models. your machine.

## Install

```bash
curl -fsSL https://peak.ai/install | sh
```

## Quick Start

```bash
peakai chat "help me debug this"
peakai agent --task "review my pr"
```

## Tech Stack

- React + Tailwind CSS
- Next.js 14
